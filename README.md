## 📞 Call App
A simple Call Notes application built using **HTML, CSS, and JavaScript**.

## ✨ Features

- ➕ Create a new call note
- 🖼️ Add profile image using Image URL
- 👤 Store full name
- 🏠 Store home town
- 📝 Add purpose of the call
- 🏷️ Select category
  - Emergency
  - Important
  - Urgent
  - No Rush
- 💾 Data stored using Local Storage
- 📇 Dynamic note cards
- ⬆️⬇️ Card navigation
- 🎨 Category color filtering (Coming Soon)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Font Awesome Icons

## 🚀 How to Run

1. Clone the repository

```bash

```

2. Open the project folder.

3. Open `index.html` in your browser.
